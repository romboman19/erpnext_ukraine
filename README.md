# ERPNext Ukraine (erpnext_ua)

Українська локалізація ERPNext: ФОП-профілі, податкові параметри, друковані форми, переклади.

## Модулі

### UA FOP
- **FOP Profile** — профіль ФОП, привʼязаний до Company: ПІБ, РНОКПП (з перевіркою
  контрольної суми), група ЄП (1/2/3), режим ставки (фіксована / 5% / 3% з ПДВ),
  IBAN, КВЕДи, адреса. Дашборд форми показує актуальні податкові параметри року.
- **UA Tax Parameters** — довідник лімітів доходу і ставок ЄП/ВЗ/ЄСВ по роках і групах.
  Записи на 2026 рік створюються автоматично при міграції (існуючі не перезаписуються).
- **UA KVED** — довідник видів діяльності.

### Податковий календар і контроль лімітів
- **UA Tax Deadline** — дедлайни ЄП/ВЗ/ЄСВ/декларацій по кожному ФОП з урахуванням
  групи; генерація з форми FOP Profile або автоматично (щомісячний scheduler).
- Щоденний scheduler: статуси дедлайнів (Скоро термін / Прострочено) і нагадування
  в Notification Log за 3 дні та при простроченні.
- Монітор доходу (касовий метод: Payment Entry + POS-оплати − повернення) проти
  ліміту групи; алерти при 80/95/100% ліміту; показник на дашборді FOP Profile.

### Друковані форми (Sales Invoice)
- Видаткова накладна (UA)
- Рахунок-фактура (UA)
- Акт виконаних робіт (UA)

Форми підтягують реквізити постачальника з FOP Profile компанії документа.

### UA Accounting

- повний український План рахунків за наказом Мінфіну №291 та чинні субрахунки
  Інструкції;
- спрощений План рахунків за наказом №186 із дозволеними операційними
  субрахунками для ERPNext;
- майстер перевірки й застосування лише до компанії без бухгалтерської або
  складської історії; діюча компанія потребує окремої міграції залишків.
- окремий позабалансовий журнал класу 0 з кількісною, вартісною та партнерською
  аналітикою, ідемпотентним API і звітом залишків без потрапляння до GL.

Нормативні джерела та порядок застосування:
[docs/ua-accounting.md](docs/ua-accounting.md).

## Сумісність
- Frappe / ERPNext v16
- Python 3.14+

## Встановлення
```bash
bench get-app https://github.com/romboman19/erpnext_ua
bench --site <site> install-app erpnext_ua
bench --site <site> migrate
```

## Roadmap
- ПРРО / фіскалізація (чеки, зміни, Z-звіти, офлайн-черга)
- Податковий календар і контроль лімітів доходу
- Мульти-ФОП owner attribution для спільного магазину
- ПДВ/ЄРПН контур для ФОП 3 групи зі ставкою 3%

Інтеграції (Нова Пошта, банки, LiqPay, POS) — в окремій app
[erpnext_ukrainian_integrations](https://github.com/romboman19/erpnext_ukrainian_integrations).
