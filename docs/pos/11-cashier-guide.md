# UA POS — коротка інструкція касира

## Початок роботи

1. Відкрити сторінку `UA POS` і ввести назву каси.
2. Відсканувати персональний штрихкод працівника.
3. Відкрити **Операції з касою → Відкрити зміну**.
4. Перерахувати купюри й монети у скриньці та підтвердити.

## Продаж

1. Відсканувати штрихкод товару або скористатися `F3` для пошуку по складу.
2. За потреби вибрати клієнта (`F4`), ідентифікувати його (`F5`) або застосувати
   знижку (`F6`).
3. Для партійного/серійного товару натиснути **Вказати** у відповідній колонці.
4. Натиснути `F9` для готівки, **Банківська картка** або **Змішана оплата**.
5. При готівці вказати отриману суму — POS покаже решту.
6. Після завершення натиснути **Друк чека**. Код унизу чека потрібен для повернення.

Якщо термінал повернув невідомий стан, не запускайте оплату повторно. Натисніть
**Перевірити стан термінала** — POS відновить попередню операцію за її idempotency key.

## Відкладені чеки

- `F7` відкладає або відновлює поточний чек.
- Кнопка **Відкладені чеки** показує всі активні кошики зміни.
- Одночасно дозволено не більше п’яти відкладених чеків.

## Повернення

1. Натиснути `F8` і відсканувати код первинного чека.
2. Вказати кількість по потрібних позиціях.
3. Розподілити виплату в межах доступних сум первинних способів оплати.
4. Провести повернення й надрукувати чек.

POS не дозволяє повернути більше проданої кількості або більше сплаченої суми.

## Касові операції та завершення дня

- **Внесення готівки** — додаткова сума у скриньку.
- **Витрата з каси** — видача з обов’язковою підставою.
- **Інкасація** — вилучення готівки зі скриньки.
- **Касовий звіт** — продажі, повернення, рух готівки та товари поточної зміни.
- Для ПРРО використовується окреме **Фіскальне меню**: відкриття, стан/X-звіт,
  Z-звіт і закриття.

Перед закриттям управлінської зміни завершіть або скасуйте всі активні чеки,
перерахуйте фактичну готівку. При розбіжності обов’язково вкажіть коментар.

## Гарячі клавіші

| Клавіша | Дія |
|---|---|
| `F2` | Фокус у поле сканування |
| `F3` | Пошук по складу |
| `F4` | Вибір клієнта |
| `F5` | Ідентифікація покупця |
| `F6` | Знижка |
| `F7` | Відкласти / відновити чек |
| `F8` | Повернення |
| `F9` | Оплата готівкою |
| `Ctrl+Alt+S` | Відкрити управлінську зміну |
| `Ctrl+Alt+C` | Закрити управлінську зміну |
